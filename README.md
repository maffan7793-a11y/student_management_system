# Student Management System

A desktop-based Student Management System developed using Python, SQLite, and CustomTkinter.

## Features

* User Login System
* Add Student
* View Students
* Search Students
* Update Student Information
* Delete Students
* Export Student Data
* Total Student Count
* Modern GUI using CustomTkinter

## Technologies Used

* Python
* SQLite3
* CustomTkinter
* Tkinter
* Git & GitHub

## How to Run

1. Clone the repository.
2. Install required libraries.
3. Run:

```bash
python login.py
```
